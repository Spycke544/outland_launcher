# 🚀 Fonctionnalités

### ⚡ Accès instantané

Lancez FiveM et rejoignez directement le serveur Outland.

### 🟢 Statut du serveur

Consultez en temps réel si le serveur est en ligne.

### 👥 Joueurs connectés

Visualisez le nombre de joueurs présents avant de vous connecter.


### 🗑️ Nettoyage automatique du cache

Supprimez facilement les fichiers temporaires FiveM afin d'éviter certains problèmes de connexion.

### 🎨 Interface moderne

Une interface élégante, rapide et intuitive pensée pour la communauté Outland.

### 🔄 Mises à jour simplifiées

Téléchargez toujours la dernière version du launcher en quelques clics.

# 📥 Installation

## Télécharger la dernière version

👉 **https://github.com/Spycke544/outland-launcher/releases/latest**

### Installation

1. Téléchargez **Outland Launcher Setup.exe**
2. Lancez l'installation.
3. Ouvrez le launcher.
4. Cliquez sur **Jouer**.
5. Profitez de votre aventure !

| Élément  | Configuration           |

# 🛠️ Fonction "Nettoyer le cache"

Le launcher peut supprimer automatiquement :

* Cache FiveM
* Logs
* Fichiers temporaires
* Cache NUI
* Server Cache

Cette opération peut résoudre de nombreux problèmes de lancement.

---

# ❓ Questions fréquentes

### FiveM n'est pas détecté

Assurez-vous que FiveM est installé et qu'il a été lancé au moins une fois.

---

### Le serveur apparaît hors ligne

Patientez quelques instants puis actualisez le launcher. Vérifiez également votre connexion Internet.

---

### Je rencontre un problème

Rejoignez le Discord afin d'obtenir une assistance rapide.
